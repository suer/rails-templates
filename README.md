Rails Application Templates
===================================

Create new apps
------------------

```
$ gem install rails
$ mkdir myapp
$ cd myapp
$ rails new . -m https://raw.githubusercontent.com/suer/rails-templates/master/template.rb
```

Main Modules
------------------

* Slim
* Devise
* Bootstrap
* RSpec
